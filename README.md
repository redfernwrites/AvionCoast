# Welcome to the Avion Coast Campaign's Documentation



### But wait, why does a *Dungeons \& Dragons* (DnD) campaign have a GitHub page???



Because the Dungeon Master (DM) of this campaign is a big friggen' nerd who wants more practice using Git and GitHub, and also wants to host a documentation website to make all the campaign lore accessible to the players! It's also a really awesome method of version control to keep track of changes that occur in the game over time. :+1:



### So what is the Avion Coast?



The Avion Coast is a homebrew campaign setting for a personal (not recorded, not streamed) long-term DnD game played by a group of friends, which started in July 2025.



### What can I expect to find in this repository?



The documentation (a.k.a. game lore) published in this repository will refer to various elements of the Avion Coast campaign setting, as well as recaps of each game session played. Information about the setting, any non-player characters (NPCs) specific to this game, and any homebrewed items, spells, classes, or house rules used by the players will be included here. Anything encountered in the game that is included in the *Players Handbook* (PHB'24), *Dungeon Master's Guide* (DMG'24), or *Monster Manual* (MM'25) may be referenced, but will not be expanded on here. However, references will be given on where to find that information, should you want to look it up.



### Can I contribute?



The inspiration for this game is very specific and has not been revealed to the players as a group (so will also not be revealed here). Feel free to comment if you want. If you feel so inclined, you *can* fork the repo and make a pull request, *however*, if you are not one of the players, your contribution has a low probability of being important enough to implement in this game. :woman\_shrugging:



### Can I used this stuff to play my own game?



Yes, under [CC BY-NC-SA 4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/deed.en). Please keep in mind that the content is meant to be incomplete and will remain incomplete in this repository, as only currently player-known information is being published here. Information exclusive to the DM within the campaign (like villain motivations, stat blocks, secret locations, etc.) will not be made available at any point. Original artwork (like maps, character and creature art, etc.) is also not planned on being published here, though that may change over time.
